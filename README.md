FinMAX - удобный чат-бот, созданный для студентов Финансового Университета При Правительстве РФ. Бот позволяет студентам просматривать расписание учебных занятий, а так же домашние задания при их добавлении прямо вместе, как в электронном журнале. Так же и преподователи могут смотреть свое раписание. 


Содержание:
- технологии
- начало работы
- Deploy
- ToDO


Технологии:
- Python3 
- maxapi
- fa_api
- sqlite3


Начало работы:
Когда директория с проектом открыта, необходимо сначала создать виртуальную среду для установки необходимых библиотек и технологий:
- python3 -m venv venv
После создания необходимо активировать виртуальную среду:
- source venv/bin/activate

После успешной активации виртуального окружения установите необходимые библиотеки из файла requirements.txt
- pip install -r requirements.txt -f

При успешной установки всех необходимых библиотек и API можно пробовать запускать проект на  локальном сервере: 
- python3 main.py


Инструкции по работе с Docker-контейнером:
- запустить docker
- docker compose build (создание контейнера)
- docker compose up -d (запуск контейнера)
- docker compose down (остановка работы)


Deploy:
Когда вы убедились, после локального запуска все работает исправно - можно запускать билд на сервере:
Для этого советуем использовать pm2, а так же после установки pm2 выполняем все те же действия, что и при запуске на локальном сервере


Объяснение работы fa_api: 
Это простая библиотека для работы с ruz.fa.ru
Для установки используем pip3 install fa_api

- Запускаем бота через команду старт
- Выбираем, что мы хотим работать с домашним заданием или с расписанием
- При выборе расписания:
- - Выбираем чье расписание мы хотим посмотреть преподавателя или групп
- - При выборе любого из расписания нужно ввести либо группу, либо преподавателя, в зависимости от вашего выбора.

- При выборе домашнего задания:
- - Выйдет выбор добавить или просмотреть домашнее задание
- - При выборе просмотреть вы выбираете группу, а так же период и если уже кто-то добавлял дз, то оно высветиться
